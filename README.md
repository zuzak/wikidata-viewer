wikidata viewer
---------------

## Configuration
In ``config.json``:
```json
{
    "restrict": true,
    "write": false
}
```

* When ``restrict`` is true, only edits to the
  [Wikidata sandbox](https://www.wikidata.org/wiki/Q4115189) will be accepted.
* When ``write`` is false, no edits will be accepted.
